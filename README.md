# Vencord Plugin for Macro Deck

A plugin for Macro Deck 2 to interact with Vencord/Vesktop via WebSocket.
This replaces the original Discord RPC plugin which doesn't support Vesktop well.

## Features
-   Mute/Deafen Toggle
-   Self Mute/Deaf status variables

## Installation
1.  Download the latest release (or build from source).
2.  Install the `.macroDeckPlugin` file.
3.  Install the **MacroDeckServer** plugin in Vencord (from the `MacroDeckServer` repository or copy).

## Configuration
-   **Port**: WebSocket port (default 8124).

## Building
Run the `create_package.ps1` script to build and package.
